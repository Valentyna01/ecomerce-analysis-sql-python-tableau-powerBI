# E-commerce Business Performance Analysis

## 📌 Project Overview
Comprehensive analysis of online store sales data to evaluate business health, calculate key performance indicators (KPIs), and provide data-driven growth recommendations.

## 🛠 Tech Stack
* **SQL:** Data extraction, joining multiple sources, and pre-processing.
* **Python (Pandas, SciPy):** Statistical analysis, Pearson correlation, and conversion rate validation.
* **Tableau & Power BI:** Interactive dashboards for business stakeholders.

## 📈 Key Insights
* **Conversion Rate:** Achieved a statistically significant **9.59%** conversion rate.
* **User Loyalty:** Registered users outnumber guests by **5x**, indicating high brand trust.
* **Top Market:** The **USA** is the leading country in both revenue and user base.
* **Top Channel:** **Organic Search** remains the primary driver of traffic.
* **Pareto Principle:** **20%** of product categories generate the majority of total profit.

## 🔬 Statistical Validation (Product Analytics)
* **A/B Test Analysis:** Verified the 9.59% conversion rate using statistical hypothesis testing.
* **Correlation Analysis:** Used Pearson correlation to identify relationships between user traffic channels and final purchase intent.
* **Platform Segmentation:** Analyzed user behavior differences between iOS and Android platforms to optimize messaging strategy.

## 💡 Strategic Recommendations
1. **Market Expansion:** Focus on European markets which show high Average Order Value (AOV).
2. **Operational Efficiency:** Increase support and warehouse staffing during weekday peak hours to improve CSAT/NPS.
3. **Product Optimization:** Conduct A/B testing specifically for the mobile app to improve mobile conversion.
4. **Future Roadmap:** Implement LTV (Lifetime Value) tracking for long-term customer analysis.
